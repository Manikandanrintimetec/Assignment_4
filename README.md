# 🧠 NLP News Article Analysis System

## 📌 Overview
This project implements a comprehensive **Natural Language Processing (NLP) system** to analyze news articles.  
It demonstrates key NLP concepts such as text processing, regular expressions, POS tagging, morphological analysis, and semantic analysis.

---

## 🎯 Features

### 🔹 1. Text Processing
- Text cleaning and normalization
- Removal of special characters
- Handling quotation marks
- Word count, character count, average word length

### 🔹 2. Regular Expression Analysis
- Extracts dates in multiple formats
- Identifies time expressions
- Detects monetary values
- Finds percentage expressions

### 🔹 3. POS Tagging
- Part-of-Speech tagging using NLTK
- Frequency distribution of POS tags
- Extraction of noun phrases and verb phrases

### 🔹 4. Morphological Analysis
- Lemmatization of words
- Identification of compound words
- Prefix and suffix analysis
- Handling irregular verbs

### 🔹 5. Semantic Analysis
- Sentiment analysis
- Named entity recognition (NER)
- Topic extraction
- Text complexity analysis

---

## 🚀 Bonus Features
- 📊 Data visualization using matplotlib
- 💻 Command-line interface (CLI)
- 📄 Support for multiple file formats:
  - `.txt`
  - `.pdf`
  - `.html`
